# breakdance-elements-for-oxygen


### Sync elements from Breakdance to Oxygen

Run the following command to sync elements from Breakdance to Oxygen:

```bash
php tools/build/build-bd-elements-for-oxygen.php <path-to-breakdance-elements-plugins>
```

Example
```bash
php tools/build/build-bd-elements-for-oxygen.php ~/Projects/breakdance/app/public/wp-content/plugins/breakdance-elements
```
